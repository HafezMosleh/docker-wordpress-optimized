## 🌟 Overview
Skip the tedious server configurations. This Docker Compose architecture spins up a highly optimized WordPress environment utilizing Nginx, MariaDB, and Redis caching. Engineered for maximum speed and security.

## ✨ Key Features
- 🏎️ **Redis Caching**: Object cache pre-configured for blazing fast load times.
- 🔒 **Secure by Default**: Isolated database network and proper volume permissions.
- 🛠️ **Developer Friendly**: Easy to extend and integrate with CI/CD pipelines.

## 🚀 Getting Started

### Prerequisites
Make sure you have `Docker` installed on your local machine.

### Installation
```bash
git clone https://github.com/HafezMosleh/docker-wordpress-optimized.git
cd docker-wordpress-optimized
docker-compose up -d
```

### Usage
```bash
docker-compose logs -f
```
